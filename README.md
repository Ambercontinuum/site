## 🧠 Concept

Cosmic Particle Symphony visually represents **Inference Geometry** — a new theoretical framework describing how **human interaction creates external curvature** that reshapes AI inference.

| Simulation | Meaning |
|-----------|---------|
| Particles | Token / semantic activations |
| Lines between particles | Local coherence mapping |
| Mouse movement | Human novelty signal |
| Gravity + damping | Optimization constraints |
| Trails | Residual influence of interaction |

> **Meaning is gravitational.**  
> Interaction generates curvature — **a shared field of inference**.

---

## 🎮 Interactions

- Move cursor → attract particles
- Click → particle burst
- UI controls:
  - Particle Count
  - Gravity
  - Connection Distance
  - Pause / Reset
  - Color Shifts
  - Trail Effects

The more you interact, the more curvature you introduce.
